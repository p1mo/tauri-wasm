# Contributing

Hey. Thank you for your interest in contributing to our Lib! We value your contributions and are excited to collaborate with you. This will help you understand how to get started and ensure that your contributions are well integrated.

## Issue Reporting

+ Report only **bugs**.
+ Questions can be asked in the `Discussions` tab under `Q&A`
+ Use the Search to check if the issue is already reported
+ All Issues will be fixed in the `dev` branch and later on merged with the `main` branch
+ Always describe clearly and in detail the steps to reproduce the issue.

## Pull Request

+ Always push to the `dev` branch

**The Naming should follow this format:**

+ New Features: `[feat] add new feature`
+ Improvements:  `[imp] improve existing feature`
+ Fixing bugs: `[fix][#1234] fix bug`
